# Security Policy

AwayLock is a local macOS utility that can lock your active session. Please treat security issues seriously.

## Reporting a Vulnerability

Open a private GitHub security advisory if available, or open an issue with minimal reproduction details and mark it as security-sensitive.

Avoid posting secrets, personal Bluetooth identifiers, or private logs publicly.

## Current Distribution Status

Preview builds are unsigned and not notarized. A production release should be signed with Developer ID and notarized by Apple before broad distribution.
